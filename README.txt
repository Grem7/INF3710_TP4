Pour bien lancer l'application:
- assurez-vous d'avoir Typescript 4.6.4 d'installé.
- assurez-vous d'avoir une base de donnée nommé: TP4 avec le mot de passe: root
- créez la base de donnée avec le fichier: AngularApp\database\bdschema.sql
- populez la base de donnée avec le fichier: AngularApp\database\data.sql
- allez dans le dossier : AngularApp\server\ et executez la commande "npm install"
- allez dans le dossier : AngularApp\client\ et executez la commande "npm install"
- allez dans le dossier : AngularApp\server\ et executez la commande "npm start" pour demarrer le serveur
- allez dans le dossier : AngularApp\client\ et executez la commande "npm start" pour demarrer le client
